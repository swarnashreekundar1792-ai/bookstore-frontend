\# Online Bookstore Application — Frontend



A React frontend for an Online Bookstore application.



\## Tech Stack

\- React 18

\- Axios (HTTP calls to Spring Boot backend)

\- Bootstrap 5 (styling)

\- react-router-dom v6 (navigation)

\- JavaScript ES6+



\## Pages

\- Book List — view all books, search, delete

\- Book Detail — view single book details

\- Add Book — form to add a new book

\- Edit Book — form to edit an existing book



\## How to Run Locally

1\. Clone this repository

2\. Run npm install

3\. Make sure the backend is running at http://localhost:8080

4\. Run npm start

5\. Frontend starts at http://localhost:3000
